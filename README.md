# MERN Shop

A modern React-based E-Commerce web application with a clean UI and responsive design.  
This project focuses on building the frontend of an online shopping platform using React.

---

## 🚀 Features

- Responsive E-Commerce UI
- Product Listing Page
- Product Details Page
- Shopping Cart UI
- Category Filtering
- React Router Navigation
- Modern Component-Based Architecture
- Clean and User-Friendly Design

---

## 🛠️ Tech Stack

- React.js
- JavaScript
- React Router DOM
- Tailwind CSS / CSS
- Vite

## ⚙️ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Rudhar-cmd/mern-shop.git
cd mern-shop
```

### 2️⃣ Install Dependencies

```bash
npm install
```

---

## ▶️ Run Project

```bash
npm run dev
```

Project will run on:

```bash
http://localhost:5173
```
